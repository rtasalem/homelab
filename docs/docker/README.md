# Docker

Below is a list of all self-hosted services that make up this homelab. Refer to the [template Docker Compose file](../../compose.template.yaml) when adding a new service to the homelab.

## Contents (A-Z)

[Bin It!](../../compose.bin-it.yaml) - Custom Python app for sending automated email alerts the day before a bin collection is due.  
[Cloudflared](../../compose.cloudflared.yaml) - Run the Cloudflare daemon (a.k.a. Cloudflared) as a service to support use of Cloudflare tunnels.  
[Navidrome](../../compose.navidrome.yaml) - Open source music server and streaming platform, an alternative to paid services such as Spotify or Apple Music.  
[Nextcloud](../../compose.nextcloud.yaml) - Alternative to Google Drive, iCloud, OneDrive etc.  
[ntfy](../../compose.ntfy.yaml) - Receive custom push notifications.  
[Ollama (with Open WebUI)](../../compose.ai.yaml) - Self-hosted AI chat for total privacy.  
[Portainer](../../compose.portainer.yaml) - GUI for monitoring Docker containers for all self-hosted services.  
[Vaultwarden](../../compose.vaultwarden.yaml) - Self-hosted password manager.

# MP3 tag editor

One small disadvantage to self-hosting a music server is that the metadata can often render inaccurate when it comes to certain properties such as year released, artist, album name etc. The solution is to use a tag editor. This does mean an extra added step, but as with all things self-hosting, tagging MP3 files allows for maximum customisation. The best website for tag editing (if you want to avoid downloading dedicated software to your machine) is [tagmp3.net](https://tagmp3.net/).
